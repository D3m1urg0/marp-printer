Install yarn,

yarn watch to build html page example

yarn tailwind to compile styles


template.pug è il file che modifichi (e che il watcher controlla per iniziare il build)

in package.json c'è un parametro nelle opzioni di marp:  pdf
se è true stampa un pdf in public
se è false aggiorna in tempo reale scheda.html (stesse grafiche, così si guarda come funziona)

è possibile montarci tailwind (già attivo ma non usato), il data è un json che esce direttamente da power app

esistono anche una versione Framework e core dello stesso prodotto (qui ho usato la CLI per fare prima)
